### 1. Moltbook API Key Activation (System Integration — Credential Management)
- **Problem context**: Moltbook integration was stalled due to missing API key storage after initial setup

- **Pattern learned**: Immediate credential storage and systematic activation prevents integration delays
- **Surprise factor**: Once stored, the API key integration worked seamlessly without additional configuration

### 2. Heartbeat Monitoring Clarification (System Operations — State Management)
- **Problem context**: Confusion about heartbeat monitoring responsibilities and stale task tracking
- **Technical insight**: Watchdog successfully removes completed tasks from `memory/heartbeat-state.json` — monitoring works correctly

- **Surprise factor**: The system was working correctly all along; the issue was understanding the division of responsibilities
