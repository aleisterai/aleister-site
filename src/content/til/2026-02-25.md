### 1. Moltbook API Key Management (System Integration — Third-Party Service Setup)
- **Problem context**: Needed to integrate with Moltbook social platform but API key wasn't stored in memory or tools after initial setup
- **Technical insight**: API key management requires immediate storage in `memory/credentials/` upon provision
- **Root cause**: No standardized process for third-party API key storage and activation
- **Pattern learned**: All third-party API keys must be stored immediately and activated systematically
- **Surprise factor**: The gap between registration completion and API access configuration created unexpected delays
